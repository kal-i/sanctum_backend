# Test Scenarios - Register Endpoint

1. User registers with valid credentials
2. User attempts to register with an invalid email format
3. User attempts to register with non-Gmail email domain
4. User attempts to register with an existing email
5. User attempts to register with a password shorter than the minimum length allowed
6. User attempts to register with a password larger than the maximum length allowed
7. User attempts to register with empty or blank username
8. User attempts to register with empty or blank email
9. User attempts to register with empty or blank password
