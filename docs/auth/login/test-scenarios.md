# Test Scenarios - Login Endpoint

1. User logs in with valid credentials
2. User attempts to log in with an invalid email format
3. User attempts to log in with an invalid password
4. User attempts to log in with an unregistered email
5. User attempts to log in with missing fields
6. User attempts to log in with empty or blank fields
7. User attempts to log in with an empty request body
8. User attempts to log in with malformed JSON
9. User attempts to log in with an unverified account
